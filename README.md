# TypeScript Backend Template

1. NodeJS + TypeScript
2. Express
3. JWT
4. Bcrypt
5. SSL
6. MongoDB
7. TypeGoose
8. Login + Sign Up + Forgot Password
9. CORS
10. BasicAuth-protected directories
11. Live Server with Hot Reloading
12. Docker + Docker Compose

### Run live-server (dev mode)
`npm run dev`

### Compile TypeScript
`npm run build`

### Deploy on Docker
`npm run deploy`
